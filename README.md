# cloud-security-posture-management

This project is a full-stack cybersecurity platform that analyzes both live cloud environments and uploaded cloud configuration files (JSON/YAML) to detect misconfigurations, score security risk using machine learning, and visualize how individual vulnerabilities can chain together into real attack paths. It combines rule-based detection mapped to industry benchmarks (CIS, NIST) with an ML layer that catches anomalies rules alone would miss, then presents everything through an interactive dashboard with compliance status and remediation guidance.

It is designed to demonstrate practical, end-to-end competency across cloud computing, cybersecurity, AI/ML, full-stack web development, and DevOps — not just one of them — by giving each domain a real, working component rather than a token feature.
